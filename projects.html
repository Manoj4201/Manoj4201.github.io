PROJECTS

Our proficiency in developing solutions covers user scenario mapping, design and development, hardware /software Integration, sub-system and system integration. These capabilities are at the core of our technology solutions portfolio with specific product solutions tailored for the defence and aerospace domain.

Pilotless Target Aircraft Ground Checkout System - Lakshya

This system is used to evaluate performance of the modules and sub modules of the Aircraft System. The software developed is used for ground check out during integration and flight trials.

Fire Control Check out - Nirbhay Flight Line Test System

Neugen developed the Health monitoring and ground checkout for validation of the Nirbhay missile fire control system. The software validates the fire control system before launch preparations.

The software validates the fire control system before launch preparations.

Compiler Test Validation Suite - Chandrayaan

A compiler verification and validation activity for the ISAC - Chandryaan Project.

Neugen carried out the complete verification and validation of the XGC ADA compiler to meet ISAC onboard software requirement standards

  • Selection of ADA95 language constructs used in a safety critical environment
  • Generation of test case matrix
  • Development of test cases to give full coverage
  • Compilation of test cases to generate assembly code
  • Test cases for linker
  • Documentation
  • Test Completion Report

Special Type Engine Test Equipment - Rustom II

Developed to operate in a laboratory for testing the Engine Instrumentation Unit (EIU) of the Rustom II It simulates all inputs to the EIU, measures and receives all ouputs, records and displays the result.

Mainly used for functional and performance tests during bench level , environmental qualification test and also during the HSI testing of the EIU The data acquisition and control application was developed in LabVIEW

Human Centrifuge - IAF

Human Cetrifuge is a closed loop control system for simulating various “G” levels for pilot in loop.

Neugen has reengineered this control system with state of the art technology using PLC for electrical units; Processor based system for control regulation.

We designed and developed various devices for the cockpit such as Pilot Physiological Monitor, Digital Video Recording System and peripheral loss of light.
